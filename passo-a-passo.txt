CONTROLE DE VERSÃO
- Mantém base histórica da evolução do código fonte de um software

Passos:
0 - Instalar o Git
1 - Escolher uma pasta que será o repositório (dir, mkdir e cd)
2 - 'git init' para iniciar um novo repositório do Git
3 - 'git status' para verificar o estado dos arquivos no repositório
4 - 'git add' para incluir arquivos no tracking do Git
5 - 'git config --global user.name "Seu Nome"'
6 - 'git config --global user.email "seu@email.com"'
7 - 'git commit -m "uma meensagem descritiva"' para confirmar as mudanças do stage para o repositório local
8 - 'git log' para ver o histórico de commits do repositório local

9 - 'git clone URL' para baixar um repositório remoto

10 - git checkout -b "Nome do braunch" (cria um novo braunch)
